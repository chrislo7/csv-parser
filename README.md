# csv-parser
CSV Parser

### Setup Instructions

1. Add desired users and passwords to `docker-compose.yml` under the environment block.
```
environment:
    MYSQL_ROOT_USER:
    MYSQL_ROOT_PASSWORD: 
    MYSQL_DATABASE: employeedb
    MYSQL_USER: 
    MYSQL_PASSWORD: 
```
